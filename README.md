# Plantilla Svelte + Vite

Plantilla inicial de Svelte para LV (bundler --> Vite)

## Para usar la plantilla:

```js
// clonar el proyecto:
git clone https://github.com/BegoMG/plantilla_svelte.git

// situarse en la carpeta del proyecto:
cd nombre_carpeta

// instalar dependencias:
npm install

// iniciar el desarrollo local:
npm run dev

// Una vez completado el proyecto, para generar los archivos de subida al servidor:
npm run build
```
